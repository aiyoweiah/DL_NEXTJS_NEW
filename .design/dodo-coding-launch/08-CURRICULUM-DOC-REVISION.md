## §1.1 — The Loop → Session structure

**v1.0 (current):**

> ### 1.1 The Loop
>
> Every session runs **Read → Reason → Speak → Decide**, in order, without exception. The verbs describe what the student is *learning the machine to do* (first three) and what only the *human* should do (fourth).
>
> | Phase | What it teaches | AI4K12 Big Idea |
> |---|---|---|
> | **Read** | How machines perceive — pixels, text, audio, sensor signals become data | 1. Perception |
> | **Reason** | How machines decide — search, logic, learning algorithms | 2. Representation & Reasoning; 3. Learning |
> | **Speak** | How machines communicate — language, generation, dialogue | 4. Natural Interaction |
> | **Decide** | What only the human should do — judge truth, fairness, intent | 5. Societal Impact + critical thinking |
>
> The Loop is the **delivery rhythm**, not the curriculum content. The pillars provide the content; the Loop sequences how it is taught in any given session.

**v1.1 (proposed):**

> ### 1.1 Session structure
>
> Every session covers **three machine abilities** the student is learning the machine to do, plus the **human discipline** of critical thinking woven throughout.
>
> | Ability | What it teaches | AI4K12 Big Idea |
> |---|---|---|
> | **Read** | How machines perceive — pixels, text, audio, sensor signals become data | 1. Perception |
> | **Think** | How machines process — search, logic, learning algorithms | 2. Representation & Reasoning; 3. Learning |
> | **Write** | How machines generate — language, output, dialogue | 4. Natural Interaction |
> | **Critical thinking** (the human discipline) | What only the human should do — judge truth, fairness, intent. Emphasized throughout every session, not isolated to a final phase | 5. Societal Impact |
>
> The session structure is the **delivery rhythm**, not the curriculum content. The pillars provide the content; the session structure sequences how it is taught in any given session.

**Rationale:** Same shape, same content, same AI4K12 mapping. Three changes:
1. Renamed: Reason → Think, Speak → Write
2. Critical thinking moves from being a "fourth phase" to a "discipline woven throughout"
3. Drops "The Loop" as the name of the rhythm

## §1.3 — Pedagogical principles

**v1.0 (current):**

> The methodology draws on five named traditions. All are cited in navigator training.
>
> - **Constructionism** (Papert, MIT) — students learn by building meaningful artifacts. Every cycle produces a tangible portfolio piece.
> - **Computational Action** (Tissenbaum, Sheldon, Abelson — MIT RAISE 2019) — students build technology that addresses real problems they care about, not synthetic exercises.
> - **Glass-box AI literacy** (AI4K12, Touretzky et al.) — show how the system works, not just what it does. Mental models before tools.
> - **Critical AI literacy** (Long & Magerko 2020; Anna Mills, MLA/CCCC task force) — prompting and AI usage taught as a form of composition; outputs are evaluated, not consumed.
> - **Socratic 1-on-1 dialogue** — inherited from DODO Learning ELA's methodology. Navigator's first move after any student answer is always a better question.

**v1.1 (proposed):** Unchanged. None of these reference The Loop. They survive intact.

## §3.5, §4.5, §5.5 — per-cycle "Loop emphasis" field

**v1.0 (current):** Every cycle (B1–B4, I1–I6, A1–A4) has a "Loop emphasis" bullet like:
> - **Loop emphasis:** Read + Reason (introduce all four phases, deepest in Read+Reason)

**v1.1 (proposed):** Field is **removed entirely** from each cycle.

**Rationale (per user direction):** The Loop emphasis field was a marketing artifact in the curriculum doc, not a Navigator-essential concept. A Navigator preparing for B1 already knows the cycle's emphasis from the existing fields: Theme, Primary pillar, Topics, AI literacy thread, Capstone, AI4K12 mastery target. The "Loop emphasis" tag was redundant labeling, not additional information.

**What replaces it for the cycles where the emphasis was genuinely informative:** Two options for the user to pick from:

- **Option A — Remove cleanly.** Just delete the field. The other fields carry the information; no replacement needed.
- **Option B — Replace with "Ability emphasis."** Same shape as Loop emphasis but using the new ability names (Read · Think · Write · Critical thinking). E.g. "Ability emphasis: Read + Think (introduce all four; deepest in Read + Think)."

**Recommendation: Option A.** The user said The Loop was a marketing artifact, not Navigator-essential. By the same logic, "Ability emphasis" is also marketing-shaped — it's the same field with a different label. If Navigators don't need a phase-tag, they don't need a phase-tag under a new name either. Default to clean removal.

If a Navigator handbook later wants explicit ability-emphasis tagging, it can be added as an internal annotation at that time, with the Navigator's input on whether it's actually useful.
